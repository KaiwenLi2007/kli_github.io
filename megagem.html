MegaGem Live

An interactive auction strategy game (1 human vs 2 AI players). Read hidden gem signals, win key auctions, and optimize final value. Built entirely from scratch using Python (Flask/Socket.IO) and Vanilla JavaScript. It features real-time multiplayer bidding, automated AI opponents, and an integrated Gemini-powered AI Tutor that offers in-game predictions and gameplay insights.

~/Landing Page ~/Live Game

Features

  • Start Game — Full game with real-time bidding, reveals, and end-round scoring.
  • Play with Tutor — Same game plus an in-game coach (Gemini) and a prediction chart of estimated gem values from public info (via QuickChart No-Code API).
  • Learn How to Play — Step-by-step tutorial and Q&A with an AI tutor.

Tech Stack

  • Backend: Python, Flask, Flask-SocketIO, eventlet
  • AI: Google Gemini (tutor + in-game coach + AI bidders)
  • Frontend: Vanilla JS, Socket.IO client
  • Charts: QuickChart API

Game Rules (Short)

  • 3 players (you + 2 AI). Everyone starts with 35 coins.
  • 30 gems (5 types: Gold, Sapphire, Jade, Onyx, Opal), 6 of each. At round start 15 are removed and split as private info (5 per player).
  • Auctions use a deck of action cards (Win 1 Gem, Win 2 Gems, Invest, Loan, etc.). Highest bid wins and pays; ties go to lower player ID.
  • When you win, you reveal one private removed gem; that info is public and affects everyone’s estimates.
  • Gem value at end of round = (total removed count for that type) × 4. Final score = coins + investment + bonus − loans + gem worth.
